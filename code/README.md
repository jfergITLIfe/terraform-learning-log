# Code

Runnable Terraform for each topic. Create folders here as you build:

```
code/
├── 03-core-workflow/
├── 04-configuration-hcl/
├── 05-modules/
├── 06-state-management/
└── 07-maintain-infrastructure/
```

Each folder should contain a `main.tf` (and any supporting files) plus a short `README.md` with the commands to run. Link from the matching topic page under `docs/topics/`.
