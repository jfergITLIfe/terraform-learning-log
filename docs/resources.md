# Resources

The things I am actually using, not a link dump.

## Primary course

- **[KodeKloud — HashiCorp Certified Terraform Associate](https://kodekloud.com/courses/hashicorp-certified-terraform-associate/)** — Spine of the study plan. Hands-on labs.

!!! note "Exam version note"
    The 003 version of this exam retired on January 8, 2026. The current version is **004** and tests on Terraform 1.12. If you're studying from the 003 course content, most material still applies, but cross-reference with the [official 004 exam content list](https://developer.hashicorp.com/terraform/tutorials/certification-004/associate-review-004) for anything that changed.

## Books

- **[Terraform: Up & Running (4th edition)](https://www.terraformupandrunning.com/)** by Yevgeniy Brikman — Read after basics click. Chapter on testing alone is worth the price.

## Practice exams

- **[Bryan Krausen's 004 practice exams](https://www.udemy.com/course/terraform-associate-practice-exam/)** — Closest thing to the real exam format.

## Official

- **[HashiCorp Learn — Terraform Associate 004 path](https://developer.hashicorp.com/terraform/tutorials/certification-004/associate-study-004)** — The official learning path, free, and genuinely good.
- **[Exam content list (004)](https://developer.hashicorp.com/terraform/tutorials/certification-004/associate-review-004)** — Definitive objective list. Use as your checklist.
- **[Sample questions](https://developer.hashicorp.com/terraform/tutorials/certification-004/associate-questions-004)** — Get a feel for the format.
- **[Terraform Registry](https://registry.terraform.io/)** — Provider and module docs. You will live here.
- **[Terraform CLI reference](https://developer.hashicorp.com/terraform/cli/v1.12.x)** — Look up flags and subcommands.

## Communities

- **[r/Terraform](https://www.reddit.com/r/Terraform/)** — Honest discussion, real problems.
- **[HashiCorp Discuss](https://discuss.hashicorp.com/c/terraform-core/27)** — Core maintainers answer questions here.
