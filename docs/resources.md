# Resources

Curated links I actually use, not a dump.

## Official

- **[HashiCorp Learn — Terraform Associate 004 path](https://developer.hashicorp.com/terraform/tutorials/certification-004/associate-study-004)** — The official learning path. Free, thorough, and genuinely good.
- **[Exam content list (004)](https://developer.hashicorp.com/terraform/tutorials/certification-004/associate-review-004)** — Definitive objective list. Use as your checklist.
- **[Sample questions](https://developer.hashicorp.com/terraform/tutorials/certification-004/associate-questions-004)** — Get a feel for the question format.
- **[Terraform Registry](https://registry.terraform.io/)** — Provider and module docs. You'll live here.
- **[Terraform CLI reference](https://developer.hashicorp.com/terraform/cli/v1.12.x)** — Flags and subcommands.

!!! note "Exam version"
    The 003 version retired on January 8, 2026. The current version is **004** and tests on Terraform 1.12. Older study material may still be accurate for fundamentals, but always cross-reference with the official 004 content list for anything that may have changed.
